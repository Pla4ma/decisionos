## What DecisionOS Is

DecisionOS is a **structured decision intelligence product** designed to help people think through important choices with clarity and confidence.

### Core Identity

- **Structured decision analyzer** — Guides users through a deliberate decision-making framework
- **Decision memory system** — Stores decisions and outcomes for future learning
- **Personal clarity tool** — Helps users understand their values and priorities
- **Outcome review loop** — Enables reflection on past decisions to improve future ones
- **First product in a future NeuroFlow ecosystem** — The foundation for broader life optimization tools

### The Decision Loop

```
Create Decision
→ Add Options
→ Answer Guided Questions
→ Gemini Analysis
→ View Scores
→ Choose Option
→ Save Decision
→ Review Outcome Later
→ Learn Personal Pattern
```

---

## What DecisionOS Is NOT

To prevent scope creep and build the right product, these are explicitly out of bounds:

| Not This | Why |
|----------|-----|
| **Generic chatbot** | DecisionOS is structured, not conversational free-form |
| **Therapy app** | No mental health counseling or crisis support |
| **Medical app** | No health diagnoses or treatment recommendations |
| **Legal app** | No legal advice or binding recommendations |
| **Investment advisor** | No financial advice or securities guidance |
| **Full NeuroFlow life OS** | MVP is decisions only; the broader ecosystem comes later |
